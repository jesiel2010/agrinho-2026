# Agro Forte, Futuro Sustentável

Projeto desenvolvido para o **Concurso Agrinho 2026 — Subcategoria 3 (Front-End)**
Tema: *"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"*

## 🌱 Sobre o projeto

Site educativo de **página única** que explica, com texto, dados e um quiz interativo, como a produção agrícola e o meio ambiente podem caminhar juntos.

🔗 **Site no ar:** https://SEU-USUARIO.github.io/SEU-REPOSITORIO/

> Substitua o link acima pelo seu link real do GitHub Pages ou Vercel **e coloque-o também na seção "About" do repositório** (obrigatório pelo regulamento).

## 📁 Estrutura

```
.
├── index.html    ← Página única (Início, Tema, Práticas, Dados, Quiz)
├── style.css     ← Todo o estilo (arquivo externo)
├── script.js     ← Toda a interatividade (arquivo externo)
└── README.md
```

## ✅ Conformidade com o regulamento (item 6.1.15)

- HTML, CSS e JavaScript **puros** — sem nenhum framework
- CSS e JS declarados em **arquivos externos** (sem inline/interno no HTML)
- Projeto **original**, sem cópia ou merge de outros repositórios
- Funciona apenas abrindo `index.html` no navegador

## ▶️ Como testar localmente

Basta abrir o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge).

## 🚀 Como publicar no GitHub + GitHub Pages

1. Crie um repositório **público** no GitHub (ex: `agrinho-2026`).
2. Faça upload dos 4 arquivos (`index.html`, `style.css`, `script.js`, `README.md`) na **raiz** do repositório.
3. Vá em **Settings → Pages**, selecione o branch `main` / pasta `/root` e salve.
4. Em alguns minutos o site estará no ar em
   `https://SEU-USUARIO.github.io/agrinho-2026/`.
5. Cole esse link funcional na seção **About** do repositório.
6. Envie o link do **GitHub** (não do Pages) pelo Ambiente Virtual de Aprendizagem em Programação até **14/06/2026 às 23h59**.
